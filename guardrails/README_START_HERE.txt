MINECRAFT-STYLE BEVY PROJECT — STABLE BASE GUARDRAILS

This archive contains the guardrails exported from the side‑scrolling project and
adapted for a full 3D / voxel-style Bevy game.

Purpose:
- Prevent structural regressions
- Separate correctness, tuning, and polish
- Establish a stable, deterministic foundation

These rules are intended to be treated as immutable until explicitly revised.
